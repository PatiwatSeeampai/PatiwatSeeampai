# Hi there, I'm Patiwat Seeampai 👋

I am an aspiring **Firmware, Test, and Automation Engineer** based in Nakhon Ratchasima, Thailand. I am passionate about bridging the gap between hardware and software, with a strong focus on Industrial Automation, Embedded AI (TinyML), and precise data acquisition systems. 

🎓 **Education:** B.Eng. in Manufacturing Automation and Robotics Engineering from Suranaree University of Technology (GPA: 3.47). 
🏆 **Honors:** Ranked 1st in the Department for three consecutive years (Academic Excellence Award 2022, 2023, 2024).

### 💼 Experience & Key Projects Highlights

* **Process Engineering Intern @ Seagate Technology:** Developed an Automated Daily Defect Dashboard using SQL, KNIME, and Python to streamline data manipulation and automated reporting.
* **AI & Computer Vision:** Trained object detection models using YOLO architecture and Roboflow for agricultural sorting, integrated with PySide6 and OpenCV.
* **Industrial IoT & Automation:** Designed a Stamping Rubber ID Tag management system utilizing ESP32, MQTT, Node-RED, and Raspberry Pi edge servers for seamless Modbus TCP data acquisition and real-time Power BI dashboards.
* **System Integration:** Built a Product Distribution System connecting PLC systems with Pymodbus, storing real-time sensor data in PostgreSQL.
